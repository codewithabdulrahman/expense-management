

# Expense Management System (core php)

# Description 

An Expense Management System is a software application designed to help organizations or individuals track, manage, and report their financial expenditures. This system aims to streamline the process of handling expenses,employeesleaves,probation ensuring that all  activities are properly recorded and easily accessible for review and analysis.

## Features

-  Manage Employees
- Manage Expense
- Manage Income
- Manage Salary
- Manage Leaves
- Manage Probation
- Manage Expense Category
- Manage Income Category
- Filter by name,title and many other etc.
- Support Multi-user level to Manage the system
- User Login & Management Included
- Custom CRUD can be added
- Easy to customize


## Installation

- Sql file will be in "Install" folder on root
- Create a database and import the sql file in your mysql database
- It has a config file that lets you store your database connection values (username, password, database name, etc.). The config file is located at application/config/database.ph
